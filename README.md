# AITranslator

#### 介绍
使用大语言模型来翻译MTool导出的待翻译文件的图像化UI软件

#### 使用说明
1.  启动TGW加载模型
2.  打开此软件
3.  选择需要翻译的文本进行翻译

#### 参与贡献
1.  Fork 本仓库
2.  新建 Develop_xxx 分支
3.  提交代码
4.  新建 Pull Request
